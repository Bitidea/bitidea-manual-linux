# 获取本机当前公网 IP 的几种方法

## 方法 1

```bash
curl https://myip.ipip.net
```

## 方法 2

```bash
curl https://cip.cc
```

## 方法 3

```bash
curl https://ipinfo.io
```

## 方法 4

```bash
curl https://realip.cc
```

## 方法 5

```bash
curl https://api.live.bilibili.com/ip_service/v1/ip_service/get_ip_addr
```

## 方法 6

```bash
curl https://ip.network/more
```
